# Neighbourhood Map

Map application that uses Google Maps API to display restaurants which includes location specific data from Facebook Places API.

Use the search bar to filter restaurants in the Aurora/Newmarket area.  Clicking on the restaurants marker will display information for the restaurant as provided by Facebook Places.

Information includes:
  - website (with link)
  - address
  - average rating and number of check ins
  - profile image
  - description (if provided to Facebook)



## Run the program locally:

1. clone the reposititory in your terminal:

```
git clone https://github.com/Courtney2511/map_project.git
```

2. Open the index file in your browser
